This project demonstrates how to connect a MySQL database to Python using SQLAlchemy and analyze the data with Pandas.

In this lab, data is retrieved from a MySQL database and loaded into a Pandas DataFrame using the read_sql() function. 
Once the data is loaded, basic exploratory data analysis (EDA) is performed to understand the dataset. 
This includes inspecting the data structure, viewing sample records, and generating summary statistics.
The lab also demonstrates how to group data by specific columns and apply aggregation functions such as sum and group by 
to analyze the data more effectively.
Overall, this project shows how Python can be used to integrate with databases and perform basic data analysis tasks efficiently.
